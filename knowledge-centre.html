🚀 OpenClaw Setup Guide

✓ What you'll achieve: A fully functional AI assistant running 24/7 on DigitalOcean, accessible from anywhere.

Part 1: Create DigitalOcean Account

1

Sign up for DigitalOcean

Go to www.digitalocean.com in your web browser

  • Click the blue "Sign Up" button in the top right corner
  • Enter your email address and create a password
  • Or sign up with Google/GitHub if you prefer
  • Click "Create Account"
2

Verify Your Email

Open your email inbox and find the verification email from DigitalOcean

  • Click the verification link in the email
  • If you don't see it, check your spam/junk folder
3

Add Payment Method

DigitalOcean requires a payment method to prevent abuse

  • Click "Billing" in the left sidebar
  • Click "Payment Methods" tab
  • Click "Add Payment Method"
  • Enter your credit card details OR connect PayPal
  • DigitalOcean will charge a small verification amount (usually $1) that gets refunded
⚠ Cost: The basic server costs $6/month. You can cancel anytime.

Part 2: Create Your Server (Droplet)

4

Create a New Droplet

A "droplet" is DigitalOcean's name for a server

  • Click the green "Create" button in the top right
  • Select "Droplets" from the dropdown menu
5

Choose Operating System

Select the software that will run on your server

  • Under "Choose an image", click "Ubuntu"
  • Select "Ubuntu 22.04 (LTS) x64"
  • LTS means "Long Term Support" - it's the most stable version
6

Choose Server Size

Select how powerful your server will be

  • Click "Shared CPU" tab (it's the cheapest option)
  • Select "Basic" plan
  • Choose the $12/month option with:
    • 2 GB RAM
    • 1 CPU
    • 50 GB SSD
💡 Tip: The $6/month option works, but $12/month gives you more breathing room for running multiple agents.
7

Choose Data Center Location

Pick where your server will physically live

  • Choose a region close to you for faster response times
  • Europe: Select "London" or "Frankfurt"
  • USA: Select "New York" or "San Francisco"
  • Asia: Select "Singapore"
8

Authentication Method

How you'll log in to your server

  • Click "Password" (easier for beginners)
  • Create a strong password (at least 12 characters)
  • Write this password down somewhere safe!
9

Name Your Droplet

Give your server a memorable name

  • Scroll down to "Choose a hostname"
  • Enter a name like: openclaw-bot or my-ai-assistant
10

Create Droplet

Launch your server!

  • Click the green "Create Droplet" button at the bottom
  • Wait 30-60 seconds while it sets up
  • You'll see a progress bar - when it's done, you'll see your droplet listed

Part 3: Install OpenClaw

11

Get Your Server's IP Address

This is how you'll connect to your server

  • On the Droplets page, find your server
  • You'll see an IP address like: 138.68.161.145
  • Click the copy icon next to it to copy it
12

Open Terminal/Command Prompt

This is where you'll type commands to control your server

  • Mac: Press Cmd+Space, type "Terminal", press Enter
  • Windows: Press Windows+R, type "cmd", press Enter
  • Linux: Press Ctrl+Alt+T
13

Connect to Your Server

Use SSH to log in to your server

  • Type this command (replace YOUR_IP with your actual IP):
ssh root@YOUR_IP

Example:

ssh root@138.68.161.145
  • Press Enter
  • Type "yes" when asked about fingerprint
  • Enter the password you created earlier
  • You won't see the password as you type - that's normal for security
✓ You're now connected! You should see something like: root@openclaw-bot:~#
14

Update System Packages

Make sure your server has the latest security updates

  • Copy and paste this command:
apt update && apt upgrade -y
  • Press Enter and wait (this takes 1-3 minutes)
15

Install Node.js

OpenClaw needs Node.js to run

  • Run these commands one at a time:
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
  • Verify it worked:
node --version

You should see: v20.x.x

16

Install OpenClaw

The moment you've been waiting for!

  • Run this command:
npm install -g openclaw
  • Wait 1-2 minutes while it installs
  • You'll see a bunch of text scroll by - that's normal
17

Start OpenClaw Setup

Configure your AI assistant

  • Run:
openclaw gateway start

You'll be asked several questions:

  • Model provider? Type: anthropic
  • Anthropic API key? Get this from: console.anthropic.com
  • Default model? Press Enter (uses Claude Sonnet)
  • WhatsApp? Type "y" if you want WhatsApp, "n" if not
18

Link WhatsApp (Optional)

Connect your WhatsApp to talk to your AI

  • A QR code will appear in your terminal
  • Open WhatsApp on your phone
  • Tap the 3 dots → "Linked Devices"
  • Tap "Link a Device"
  • Scan the QR code on your computer screen
  • Wait for "Connected!" message
🎉 Congratulations! OpenClaw is now running 24/7. Send a WhatsApp message to test it!

Troubleshooting

Problem: Can't connect via SSH
  • Make sure you copied the IP correctly
  • Wait 2-3 minutes after creating droplet
  • Try adding -o StrictHostKeyChecking=no to the ssh command
Problem: Password doesn't work
  • Remember: you won't see characters as you type
  • Make sure Caps Lock is off
  • Reset password in DigitalOcean console if needed
Problem: OpenClaw won't start
  • Run: openclaw status to see what's wrong
  • Check logs: openclaw gateway logs
  • Make sure API key is correct

🎯 Control Room Roadmap

✓ What you'll build: A professional AI operations dashboard with 8 specialized agents, task management, live monitoring, cost tracking, and video generation.

Overview: What is the Control Room?

The Control Room is a web-based dashboard that lets you manage multiple AI agents working together. Think of it as "Mission Control" for your AI workforce.

Task Management

Create, assign, and track tasks across 4 pipeline stages

8 Specialized Agents

Google Ads, Meta Ads, Lead Gen, News, Email, Video, Script, YouTube

Live Feed

Real-time updates on agent activity and system events

Cost Tracking

Monitor API usage and costs per agent/task

Phase 1: Setup Foundation

1

Create Project Directory

Connect to your server and create the workspace

cd ~/.openclaw/workspace mkdir mission-control cd mission-control
2

Install Dependencies

Install the Node.js packages we'll need

npm init -y npm install express cors body-parser chokidar

What each package does:

  • express: Web server for the dashboard
  • cors: Allows requests from different origins
  • body-parser: Processes incoming data
  • chokidar: Watches files for changes
3

Create Data Storage Files

These JSON files will store your data

touch tasks.json live-feed.json cost-tracking-data.json agent-models.json

Initialize each file with empty data:

echo '[]' > tasks.json echo '[]' > live-feed.json echo '{"totalCost": 0, "totalCalls": 0, "agents": {}}' > cost-tracking-data.json echo '{}' > agent-models.json

Phase 2: Build the Backend

4

Create API Server

This is the heart of your Control Room - it serves the dashboard and handles data

Create server.js with these features:

  • REST API endpoints for tasks, live feed, cost tracking
  • Static file serving for the dashboard HTML/CSS/JS
  • Binds to 0.0.0.0:8086 (accessible from anywhere)
  • CORS enabled for cross-origin requests

Key endpoints:

GET /api/tasks - List all tasks POST /api/tasks - Create new task PUT /api/tasks/:id - Update task GET /api/live-feed - Get recent events GET /api/cost-tracking - Get cost data PUT /api/agent-model - Update agent model
5

Create Task Monitor

Watches tasks.json and logs changes to live feed

Create task-monitor.js that:

  • Uses chokidar to watch tasks.json for changes
  • Detects new tasks, updates, completions
  • Logs events to live-feed.json
  • Runs in background 24/7
6

Start the Servers

Launch both services

node server.js > server.log 2>&1 & node task-monitor.js > task-monitor.log 2>&1 &

Verify they're running:

ps aux | grep node
✓ You should see both server.js and task-monitor.js running

Phase 3: Build the Dashboard UI

7

Create Main Dashboard

Create index.html with:

8

Add Task Creation Modal

Build the "Create Task" interface

  • Modal popup with form fields
  • Title, description, priority (Low/Medium/High)
  • Agent assignment dropdown
  • Tags input (optional)
  • Submit → POST /api/tasks → refresh dashboard
9

Implement Live Feed

Right sidebar showing real-time activity

  • Auto-refreshes every 2 seconds
  • Color-coded by event type:
    • 🟢 Green: Task completions
    • 🔵 Blue: New tasks
    • 🟡 Yellow: Updates/changes
    • 🔴 Red: Errors/warnings
  • Shows last 20 events
  • Timestamps in relative format ("2 min ago")
10

Add Cost Tracking Page

Monitor API usage and expenses

  • 4 stat cards: Total Cost, API Calls, Active Agents, Budget Remaining
  • Per-agent breakdown table
  • Per-task cost tracking
  • Progress bar showing budget usage

Phase 5: Automation & Scheduling

20

Set Up Cron Jobs

Schedule agents to run automatically

Use OpenClaw's built-in cron system:

openclaw cron add \ --name "Morning Briefing" \ --schedule "0 7 * * *" \ --task "Run Agent 5: AI News Scout"

Set up cron jobs for:

  • 7:00 AM - Morning AI news briefing
  • 9:00 AM - Lead gen campaign (5 emails)
  • Every hour - Inbox monitoring
  • 10:00 AM - YouTube AI tips upload
  • 11:00 AM - YouTube news summary upload
21

Configure Agent Communication

Agents can trigger each other

  • Script Writer completes → triggers Video Wizard
  • Video Wizard completes → triggers YouTube Agent

Implement via webhook system or shared task queue
